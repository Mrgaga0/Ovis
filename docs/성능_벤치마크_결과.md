## 3. 성능 지표

### 3.1 Core Web Vitals
- **LCP (Largest Contentful Paint)**
  - 평균: 1.8초
  - 90퍼센타일: 2.5초
  - 목표 달성률: 95%

- **FID (First Input Delay)**
  - 평균: 20ms
  - 90퍼센타일: 50ms
  - 목표 달성률: 98%

- **CLS (Cumulative Layout Shift)**
  - 평균: 0.05
  - 90퍼센타일: 0.1
  - 목표 달성률: 99%

### 3.2 로딩 성능

  - 번들 크기: 120KB (gzip)
  - 캐시 히트율: 95%
